	Door Panel Tolerance Parameters Monitor

1. Input "EXIT" to quit application.
2. Input "SHOW" to show all the parameters of each area.
3. Input "SHOW [AREA]" to show specific area's parameters.
4. Input "AREALIST" to show all areas' name.
5. Input "MONITOR [AREA]" to monitor area's tags.
6. Input "CLEAR" to clear screen.
7. Input "CMD" to show command list.
