	Door Panel Queue Service

1. Input "EXIT" to quit application.
2. Input "SHOW" to show all content of queue.
3. Input "SEND ORDER PARAM" to send specified order's tooling parameter value.
4. Input "SEND TOOL PARAM" to send specified tooling parameter value.
5. Input "START" to start service.
6. Input "STOP" to stop service.
7. Input "CLEAR" to clear screen.
8. Input "CMD" to show command list.
